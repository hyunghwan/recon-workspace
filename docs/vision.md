# Recon Workspace — Vision

## One-line pitch
A reconciliation workspace for bookkeepers and small finance teams to match statements to supporting documents, track exceptions, and close books faster.

## Initial wedge
Start with one painful job:

**Find unmatched transactions and chase missing supporting documents.**

Not a full accounting system.
Not a replacement for QuickBooks or Xero.
Not an expense-report tool.

It lives in the messy gap between raw statements and clean bookkeeping.

## Target user
### Primary
- Outsourced bookkeepers
- Small bookkeeping agencies
- Fractional controllers serving SMBs

### Secondary
- Small finance/ops teams at startups
- Ecommerce operators who do their own bookkeeping prep

## Core pain
Users already have:
- bank/card statements
- receipts, invoices, screenshots, PDFs, CSVs
- accounting software

But they still spend too much time on:
- matching transaction lines to support docs
- identifying missing receipts/invoices
- following up for missing documentation
- tracking unresolved exceptions before month-end close
- remembering why a line item is still unresolved

## Why this matters
The work is:
- repetitive
- error-prone
- hard to delegate cleanly
- expensive in cumulative monthly time

## Product thesis
If we make exception handling and missing-doc chasing dramatically cleaner, users will pay even before deeper automation exists.

## Positioning
### What we are
- reconciliation prep workspace
- exception tracker
- supporting-doc matching layer

### What we are not
- ERP
- general ledger
- tax platform
- close management suite for enterprise

## Product principles
1. Start with the unresolved items, not the happy path.
2. Every transaction should have a clear state.
3. Human review should feel fast, not buried.
4. Audit trail beats cleverness.
5. The first version should save time on day one.

## Success criteria for v1
A bookkeeper should be able to:
- upload a statement or transaction file
- see unmatched items quickly
- assign or attach support docs
- mark items as missing docs / matched / needs review
- export or summarize unresolved exceptions for follow-up
