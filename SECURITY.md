# Security Policy

## Supported scope
Recon Workspace is a public reference app, not a hardened enterprise product. Security fixes are still welcome, especially around authentication, data isolation, file uploads, and Firebase rules.

## Reporting a vulnerability
- Do not open a public issue with working exploit details.
- Use GitHub private vulnerability reporting for this repository if it is enabled.
- If private reporting is not available, open a minimal issue asking for a private contact path without including exploit steps or secrets.

## What helps a report
- A clear description of the impact
- Reproduction steps
- The affected file, route, or Firebase rule if known
- Any assumptions required for the issue to be exploitable

## Response expectations
This repository is maintained on a best-effort basis. There is no formal SLA, but legitimate reports will be reviewed as capacity allows.
